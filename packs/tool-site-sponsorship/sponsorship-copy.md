# Tool-site Sponsorship — tiers + pricing + CTA copy

**Sites (examples):** btw-calculator.be, kmvergoeding.be, datumberekenen.be, kleurcodes.be, ibanvalidator.be, …

## Goal
Create a simple, repeatable sponsorship offer that can be:
- shown on the tool sites (“Adverteer hier”)
- linked to a lightweight landing section/page
- sold as a monthly subscription

---

## 3 Sponsorship tiers

### Bronze — “Featured Mention”
**Best for:** local businesses + solo operators who want visibility on a niche tool page.

**Includes**
- 1 featured mention block (text + logo)
- 1 dofollow link (or nofollow if requested)
- Shown on **1 tool site**
- 1 revision/month

**Placement ideas**
- Under the calculator result
- Sidebar / inline block between sections

**Suggested price**
- **€250 / month**

---

### Silver — “Sponsored Slot”
**Best for:** companies who want consistent lead flow + better positioning.

**Includes**
- Everything in Bronze
- Better placement (above the fold where possible)
- 2 variants A/B (headline + CTA)
- Shown on **up to 3 tool sites**
- Monthly click/report summary (simple)

**Suggested price**
- **€500 / month**

---

### Gold — “Exclusive Sponsor”
**Best for:** brands who want category ownership (max exposure, less competition).

**Includes**
- Everything in Silver
- **Exclusivity** (only 1 sponsor in that slot)
- Shown on **up to 8 tool sites**
- Priority placement + custom copy per site
- Quarterly review call / refresh

**Suggested price**
- **€1000 / month**

## CTA copy (tool-sites)

### Short CTA (button)
- **Adverteer hier**
- **Sponsor deze tool**
- **Plaats je advertentie**

### Micro-copy (1–2 lines)
- “Bereik maandelijks duizenden bezoekers die deze tool gebruiken. Sponsor deze pagina.”
- “Wil je hier staan? Promoot je business op deze toolpagina.”
- “Sponsoring vanaf €250/maand. Snel geregeld.”

### CTA block (ready-to-paste)
**Titel:** Adverteer op deze tool

**Tekst (NL):**
> Deze tool wordt elke maand door duizenden mensen gebruikt. Wil je hier zichtbaar zijn met een sponsorblok + link?
> Sponsoring start vanaf **€250/maand**.

**Knop:** Adverteer hier

**Tekst (EN):**
> This tool is used by thousands of visitors every month. Want a featured sponsor block + link on this page?
> Sponsorship starts at **€250/month**.

**Button:** Advertise here

## FAQ (tiny)
**Is dit Google Ads / AdSense?**
Nee. Dit is een vaste sponsorship (direct placement) op de pagina.

**Kunnen we wisselen per maand?**
Ja, maandelijks opzegbaar (tenzij anders afgesproken).

**Krijg ik rapportage?**
Silver/Gold krijgen een simpele maandelijkse samenvatting (kliks/period).
